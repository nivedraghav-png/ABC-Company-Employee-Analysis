# 📊 ABC Company Employee Data Analysis

This repository contains my **Well Executed Project** for Python.  
The goal is to analyze an employee dataset from ABC Company and generate insights.

---

## 🚀 Objectives
- Clean and preprocess the dataset (fix height column).  
- Perform data analysis to answer 5 key business questions.  
- Create visualizations to support the findings.  
- Provide a data story summarizing insights.  

---

## 📝 Key Analyses
1. **Employee Distribution by Team** – Counts & percentages.  
2. **Segregation by Position** – Employee roles across the company.  
3. **Predominant Age Group** – Workforce demographics.  
4. **Highest Salary Expenditure** – By team × position.  
5. **Age vs Salary Correlation** – Trend analysis with scatterplot.  

---

- `Module1_End_Project.ipynb` → Jupyter Notebook with code & outputs.
